# Turing Website

Turing 的官方网站，使用现代 Web 技术栈构建。

## 技术栈

- **Nuxt 3** — Vue.js 全栈框架
- **Nuxt Content v3** — 文件驱动的 CMS
- **Tailwind CSS** — 原子化 CSS 框架
- **Vercel** — 静态部署

## 特性

- 全站 SSG，极致性能
- 暗色 Deep Tech 视觉风格
- 响应式设计
- SEO 优化
